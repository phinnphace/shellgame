# shellgame 0.1.0

* Initial release.
* Provides an accessible, user-facing contract to evaluate geographic crosswalk transformations.
* Introduces the `evaluate_transformation()` wrapper, which simplifies the application of the `geoDeltaAudit` engine into a single function call.
* Uses "perturbation" terminology to correctly identify transformation loss as a mathematical reality of spatial alignment schemas rather than user data error.
* Includes narrative vignettes demonstrating the workflow using Hennepin County as a case study.
